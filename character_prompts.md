# Character Illustration Prompts — Technology Explorer Series

**Style Direction (apply to ALL prompts):**
> Friendly, rounded children's book illustration style. Bold outlines, vibrant flat colours,
> simple shading. Full-body character on a white/transparent background. Anthropomorphic but
> clearly representing the technology. Suitable for ages 8–12. No text in image.
> Consistent proportions: characters should be roughly the same height (except where noted).
> Think: Pixar meets educational poster.

---

## Global Style Suffix
Append this to every prompt:

```
children's educational book illustration, full-body character design, bold clean outlines,
flat vibrant colours, friendly expression, white background, no text, high resolution,
vector-style, suitable for ages 8-12
```

## Book 1 — The Secret Life of the Internet
Primary palette: #2196F3 (InternetBlue), #FF9800 (WarmOrange), #4CAF50 (LeafGreen)

### Dot
```
A tiny, round, glowing orb character with a warm golden-yellow light, about the size of a
marble. Has two small cheerful eyes and a bright smile. Leaves a faint sparkle trail behind.
Wears a miniature envelope-shaped backpack with a numbered label "001" on it. Body is
translucent and softly luminous, like a firefly. Represents a data packet.
```

### Bee Browser
```
A cheerful cartoon bee wearing a tiny pilot's cap and goggles pushed up on forehead. Wings
are shaped like browser tab icons. Body is striped in blue (#2196F3) and white instead of
yellow and black. Carries a small compass in one hand. Friendly wide smile. Represents a
web browser — a guide that takes you to websites.
```

### Willa Wi-Fi
```
A graceful, translucent wave-shaped character made of flowing curved lines, coloured in soft
blue and purple gradients. Has gentle eyes in the centre of the wave pattern. Body ripples
outward like concentric pond ripples. Arms are gentle arcs of signal. Floats slightly off
the ground. Represents invisible Wi-Fi radio waves.
```

### Rory Router
```
A friendly, boxy character shaped like a home router — rectangular body with rounded corners,
two small antenna "ears" on top that wiggle. Has a row of blinking LED dots across the chest
like buttons on a shirt. Colour: warm orange (#FF9800) with white accents. Short sturdy legs.
Wears a postmaster's cap. Represents a home router directing network traffic.
```

### Sunny Server
```
A tall, warm-golden character shaped like a server rack — rectangular and upright with
horizontal shelf lines across the body like ribs. Glows with a soft golden light. Has kind,
wise eyes near the top. Small arms emerge from the sides holding folders and files. Stands on
a cooling fan base that acts as feet. Represents a powerful server computer.
```

### Captain Cable
```
A long, rope-like serpentine character with a tough, layered body showing cross-section rings
(dark outer shell, copper middle, glowing fibre-optic core visible at the tip). Wears a sea
captain's hat. Has a weathered but friendly face with a thick moustache. Tail end disappears
into ocean waves. Colour: dark grey/navy outer, orange copper mid-layer, bright cyan glow at
core. Represents undersea internet cables.
```

### Scout Search
```
A small, energetic detective character — magnifying glass as the head (lens is the face with
bright curious eyes). Body is a compact spider-like shape with eight nimble legs (referencing
web crawlers). Wears a tiny detective hat. Colour: deep blue body with green highlights.
Carries a small notebook. Quick, alert posture. Represents a search engine.
```

### Cloud Pal
```
A soft, puffy cloud character floating in the air with a friendly smiling face. Inside the
semi-transparent cloud body, tiny glowing server racks are faintly visible. Has small arms
made of wisps of cloud. Wears a tiny padlock necklace (representing security). Colour: white
and soft sky blue with subtle purple undertones. Represents cloud computing/storage.
```

## Book 2 — Inside a Smartphone
Primary palette: #7B1FA2 (TechPurple), #FF9800 (WarmOrange), #4CAF50 (LeafGreen)

### Chip
```
A small, square character shaped like a microprocessor chip — flat rectangular body with tiny
gold pin-legs along all four edges (like a centipede). Surface has a circuit-board pattern
etched across it. Eyes are two bright LED-like dots. Colour: deep purple (#7B1FA2) with
gold traces. Wears a tiny crown (brain of the phone). Arms are flexible circuit traces.
Energetic, fast-moving posture. Represents the phone's processor.
```

### Memo
```
A character split into two halves — left side is a fast, glowing stick of RAM (tall, thin,
electric blue with pulsing lights), right side is a sturdy filing cabinet-shaped storage
block (wider, silver-grey, with tiny labelled drawers). Both halves share one friendly face
in the middle. Holds a photo album in one hand and a sticky note in the other.
Represents memory and storage.
```

### Tappy
```
A flat, rectangular character shaped like a phone screen — glass-smooth surface body with a
faint grid pattern visible beneath. Face appears ON the screen (like an emoji). Fingers/hands
are made of ripple-rings (like touch ripples on water). Body glows where touched. Colour:
black frame with a vibrant display surface showing colour gradients. Represents the
touchscreen.
```

### Lens
```
A round, camera-lens-shaped character — circular body with concentric glass rings forming the
face. One large "eye" in the centre (the aperture) that can widen and narrow. Small flash
unit on top of head like a hat. Body is sleek dark grey with purple lens flare accents.
Holds a tiny photo frame. Represents the phone's camera system.
```

### Zap
```
A battery-shaped character — cylindrical/rectangular body with a + symbol on the head and –
on the feet. Colour shifts from green (full) at the top to red (empty) at the bottom in a
gradient. Has lightning-bolt shaped arms. Energetic but sometimes tired expression. Wears
tiny running shoes. A small lightning bolt icon floats above the head.
Represents the phone battery.
```

### Sensa
```
A compact, multi-tool character with a spherical body covered in tiny sensor "eyes" — each a
different colour representing different sensor types (red for infrared, blue for gyroscope
motion, green for accelerometer arrows, yellow for compass needle). Has a spinning gyroscope
ring orbiting the body. Alert, attentive expression. Represents the phone's hidden sensors.
```

### Appy
```
A character shaped like a rounded-square app icon — body is a colourful rounded rectangle
with a dynamic, shifting pattern on the surface (showing mini app icons). Has stretchy arms
and legs. Wears a tiny toolbelt with miniature icons (music note, map pin, camera, game
controller). Cheerful, versatile expression. Colour: rainbow gradient frame.
Represents smartphone apps.
```

### Orbi
```
A small satellite character orbiting in space — boxy central body with two large solar-panel
wings extending from each side. Has a small dish antenna on top like a hat. Body is silver
and blue. Floats with small orbit rings drawn around it. Friendly face on the main body
panel. Earth is tiny in the background below. Represents GPS satellites.
```

## Book 3 — Robot World
Primary palette: #00C853 (RoboGreen), #FF9800 (WarmOrange), #607D8B (SteelBlue)

### Bolt
```
A large industrial robot arm character — single powerful arm mounted on a heavy base. The arm
has multiple joints (shoulder, elbow, wrist) with visible bolts. Gripper hand is open in a
friendly wave. Body is bright orange (#FF9800) with steel-grey joints. Has a simple face on
the "shoulder" joint — two round eyes and a determined smile. Sparks fly from the
welding-tip fingertip. Represents factory/industrial robots.
```

### Whirr
```
A quadcopter drone character — compact body with four propeller arms extending outward.
Propellers are spinning (motion blur). Has a camera "eye" on the underside. Main body is
white and green (#00C853). Friendly face on the front panel. Small package held underneath
in tiny claws. Tilted slightly forward as if flying. Represents delivery/surveillance drones.
```

### Sprocket
```
A six-wheeled rover character — low, wide, sturdy body on six chunky wheels (like Mars
rover). Has a camera mast "neck" extending up with a curious face on top. Body is dusty red
and steel grey. Solar panel on the back like a backpack. One mechanical arm extends forward
holding a rock sample. Rugged, adventurous appearance. Represents exploration rovers.
```

### Cora
```
A friendly humanoid robot character — smooth white body with rounded joints (no sharp edges).
About child-sized proportions. Face is a simple curved screen showing two gentle blue eyes
and a warm smile. Hands are soft/padded (safe for interaction). Wears a small scarf. Body
has subtle green (#00C853) accent lines at joints. Standing with one hand extended in
greeting. Represents companion/social robots.
```

### Sparks
```
A tiny micro-robot character — extremely small (shown with a coin for scale). Sleek,
capsule-shaped body with precise miniature surgical tools extending like arms (tiny forceps,
laser, camera). Body is medical white and steel blue. Has a focused, careful expression with
tiny magnifying-glass eyes. A subtle green cross on the chest. Represents medical/surgical
micro-robots.
```

### Tread
```
A heavy-duty rescue robot character — tank-like treaded base with a powerful upper body.
Thick armoured plating. Has a searchlight on top of the head and a camera "eye". One arm is
a heavy-lift grabber, the other has a water cannon. Colour: bright rescue-orange and black.
Tough but kind face. Shown bracing against rubble. Represents search-and-rescue robots.
```

### Echo
```
A smooth, egg-shaped home assistant robot character — rounded body on a small wheeled base.
Top of the head has a glowing ring (like a smart speaker light ring) that pulses blue when
listening. Simple friendly dot-eyes and a curved smile. Small screen panel on the chest
showing a waveform. Colour: white body with soft blue (#607D8B) accents. Calm, attentive
posture. Represents voice-controlled home assistant robots.
```

## Usage Notes

1. **Consistency**: Generate all characters in one session if possible, or reference earlier outputs to maintain style consistency.
2. **Colour accuracy**: Use the hex codes provided to keep characters on-palette with their book's colour scheme.
3. **Transparency**: Request PNG with transparent background for LaTeX integration.
4. **Resolution**: Minimum 300 DPI for print, 150 DPI acceptable for digital.
5. **Variants needed per character**:
   - Full-body standing (primary — for "Meet Your Guides" page)
   - Small icon/badge version (for chapter headers and callout boxes)
   - Action pose (for specific chapter appearances)
